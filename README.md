# Leave Tracking System (Java Backend)

A Java-based backend system to manage employee leave requests,
including sick leave validation, approval workflow, and status tracking.

## Features
- Employee management
- Leave request tracking
- Sick leave validation rules
- Approval queue using Java Collections
- OOP principles (Abstraction, Inheritance, Interface)

## Tech Stack
- Java
- OOP
- Java Collections Framework

## Future Enhancements
- Spring Boot REST APIs
- Database integration (MySQL)
- Role-based access
- JUnit testing
